# Splash简介

Splash是为Scrapy爬虫框架提供渲染javascript代码的引擎，它有如下功能：（摘自维基百科）

（1）为用户返回渲染好的html页面

（2）并发渲染多个页面

（3）关闭图片加载，加速渲染

（4）执行用户自定义的js代码

（5）执行用户自定义的lua脚步，类似于无界面浏览器phantomjs

# splash demo

import requests
from lxml import etree

keyword = "笔记本电脑"
params = dict(
    keyword=keyword,
    enc='utf-8',
    wq=keyword,
    pvid='e2d34e8a208e49518630154bb4d05307'
)
headers = {
    'user-agent': 'Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/65.0.3325.181 Safari/537.36'
}
query_string = '&'.join('%s=%s' % (k, v) for k, v in params.items())
jd_url = 'https://search.jd.com/Search?' + query_string
# SPLASH_URL搭建splash的服务器url地址
url = 'http://SPLASH_URL:8050/render.html?url=' + jd_url
print(url)
r = requests.get(url, headers=headers)
selector = etree.HTML(r.text)
prices = selector.xpath('//li[@class="gl-item"]//strong/i/text()')
names = selector.xpath('//div[contains(@class, "p-name")]/a/em/text()')

for name, price in zip(names, prices):
    print(name, price)
