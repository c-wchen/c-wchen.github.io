Scrapy是一个用于爬网网站并提取结构化数据的应用程序框架，可用于各种有用的应用程序，例如数据挖掘，信息处理或历史档案。

即使Scrapy最初是为Web抓取而设计的，它也可以用于使用API（例如Amazon Associates Web Services）或用作通用Web搜寻器来提取数据

# quotes_spider.py
import scrapy
class QuoteSpider(scrapy.Spider):
    name = 'quote'
    # url
    start_urls = ['http://quotes.toscrape.com/tag/humor/']

    def parse(self, response):
        quotes = response.xpath('//div[@class="quote"]')
        for quote in quotes:
            yield {
                'author': quote.xpath('span/small/text()').extract_first(),
                'title': quote.xpath('span[@class="text"]/text()').extract_first()
            }
        # 分页中下一页
        next_page = response.xpath('//li[@class="next"]/a0h0ref').extract_first()
        if next_page:
            yield response.follow(next_page, self.parse)

运行代码

# 创建项目
scrapy startproject PROJECT_NAME
# 生成spider文件按
scrapy genspider  SPIDER_NAME DOMAIN_NAME # -指定模板
# 运行爬虫
scrapy crawl SPINDER_NAME -o quotes.json
# 单独运行爬虫文件
# 运行并保存内容为json格式,只需要py文件即可运行
scrapy runspider SPIDER_NAME.py -o quotes.json
# 保存为csv格式
scrapy runspider SPIDER_NAME.py -o quotes.csv -t csv

scrapy shell

# Scrapy终端是一个交互终端，我们可以在未启动spider的情况下尝试及调试代码，也可以用来测试XPath或CSS表达式，查看他们的工作方式，方便我们爬取的网页中提取的数据,如果安装ipython会使用ipython的终端

scrapy shell内置函数

函数 说明
fetch 请求url或者Request对象， 注意， 请求成功之后会自动将当前域内的request和response对象重新赋值
view 用浏览器打开response对象内的网页
shelp() 打印出帮助信息
spider 相应的Spider对象的实例（default|SPIDER）
setting 保存所有配置信息的对象
crawler 所有内置函数的集合 （crawler.setting is setting ）True
